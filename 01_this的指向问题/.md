# this的认识
1. 函数在调用时会默认绑定一个this，this是在运行时被绑定的
2. this的绑定和编写位置无关，和调用方式有关

# 默认绑定几种调用方法
```js
//1. 普通函数，独立调用
function foo() {
  console.log('foo this :', this)
}
foo()

//2. 定义在对象种，但是独立调用
var obj ={ 
  name: 'why',
  bar: function () {
    console.log('bar this:', this)
  }
}
var baz = obj.bar
baz()

//3. 高阶函数
var obj1 = {
  bar: function() {
    console.log('obj1 bar this:', this)
  }
}
function test(fu){
  fun()
}
test(obj1.bar)
```
